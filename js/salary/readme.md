## js 数据类型
    简单数据类型
        - number
            不分长整型 短整型 浮点 都是数值类型number
        - string
        - boolean
        - null      定义了但值为空
            为空，值为空
        - undefined 未定义  
            声明了变量（只声明），但是没有赋值
            内存空间（undefined 有地址， 但是地址的长度不定） 弱类型语言
    复杂数据类型
        - object
            除了简单数据类型  都是对象
            array [] 可遍历的对象
            object {} 对象字面量
            function   函数是可执行的对象
    -自学作业   typeof 
        可以获得变量的数据类型，但是不准确
